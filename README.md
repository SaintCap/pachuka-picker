# 🎮 ПАЧУКА // Терминал выбора игры

Статический сайт для выбора игры: пользователь читает зашифрованные описания,
выбирает одно, жмёт «Выбор сделан» — и с эффектами узнаёт название игры
и получает ссылку на её страницу в Steam.

## Структура проекта

| Файл        | Что это                                                        |
|-------------|----------------------------------------------------------------|
| `games.js`  | **Конфиг игр — единственный файл, который нужно редактировать** |
| `index.html`| Разметка страниц                                               |
| `style.css` | Оформление и анимации                                          |
| `app.js`    | Логика, частицы, звуки, эффекты                                |

## Как добавить/изменить игры

Открой `games.js` и отредактируй список. Одна игра = один блок:

```js
{
  description: "Загадочное описание игры (название не указывай!)",
  name: "Название игры",
  steamUrl: "https://store.steampowered.com/app/XXXXXX/Название/"
}
```

Ссылку на Steam бери прямо из адресной строки страницы игры в магазине.
Игр может быть сколько угодно. После правок — commit + push, и Render
задеплоит новую версию автоматически.

## Деплой на Render.com

Сайт полностью статический — база данных и сервер не нужны.

1. Залей папку проекта в репозиторий на GitHub (или GitLab):
   ```bash
   git init
   git add .
   git commit -m "Pachuka game picker"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_ЛОГИН/pachuka-picker.git
   git push -u origin main
   ```
2. Зайди на [render.com](https://render.com) → **New +** → **Static Site**.
3. Подключи свой репозиторий.
4. Настройки:
   - **Build Command:** оставь пустым
   - **Publish Directory:** `.` (точка — корень репозитория)
5. Нажми **Create Static Site** — через минуту сайт будет жить на
   `https://имя.onrender.com`.

Каждый push в `main` автоматически передеплоивает сайт.

## Локальный запуск

Просто открой `index.html` в браузере, либо:

```bash
python3 -m http.server 8000
# открой http://localhost:8000
```

## Фишки

- Кастомный неоновый курсор со шлейфом частиц
- Глитч-заголовок, летящие звёзды, ретро-сетка в стиле synthwave
- 3D-наклон карточек за мышкой
- Взрыв частиц, вспышка и тряска экрана при раскрытии
- Эффект «расшифровки» названия игры (scramble-текст)
- Синтезированные звуковые эффекты (Web Audio, без файлов)
- Адаптив под мобильные + уважение к `prefers-reduced-motion`
